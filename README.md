# market-analysis-env
Market tools setup (jupyter and visual studio code)
## Usage
### Installation
```powershell
\build.ps1
```

### Run
```powershell
\run.ps1
```

Then in your browser

* Visual Studio code service
```
http://localhost:8080/
```
The password is 'market'

* Jupyter Lab service
```
http://localhost:8888/lab
```
